# NORDSTERN-BANK-ADMIN-
Bank admin system with C#, MySQL and Python FastAPI. Includes login system, API backend and database integration

This project is a self-built admin system for a banking application.

It combines C#, MySQL and Python FastAPI to simulate real backend and database operations.



The goal was to understand how real systems handle authentication, data management and API communication.


Login system with database validation

REST API built with FastAPI

Database integration using MySQL

Admin dashboard for managing users/data

Backend logic implemented in C#
